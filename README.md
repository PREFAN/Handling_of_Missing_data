# Handling_of_Missing_data
This project explores various techniques for handling missing values in machine learning using the California Housing dataset. It begins by demonstrating the problem of missing data through visualization (heatmap and bar charts) and shows that scikit-learn models cannot train directly on datasets containing null values. The code then implements and compares five different approaches: dropping rows with missing values, mean imputation, median imputation, KNN imputation, and Multiple Imputation by Chained Equations (MICE). An interactive widget-based dashboard allows users to switch between these methods in real-time, observing changes in data distribution, model performance (MSE and R²), prediction accuracy, and feature importance. The project effectively highlights the trade-offs between data loss and data quality when dealing with missing values.
